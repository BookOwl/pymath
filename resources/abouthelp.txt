PyMath
Version 0.1

About:
PyMath is a simple program designed to help math students (a.k.a, me) solve problems.

Currently, it can:
Factor and simplify expressions
Solve single-variable equations
Plot expressions
I plan to add more features soon.

The homepage for PyMath is https://github.com/BookOwl/pymath

Help:
If you are having any trouble with PyMath, or if you discover a bug, please create an issue at https://github.com/BookOwl/pymath/issues
